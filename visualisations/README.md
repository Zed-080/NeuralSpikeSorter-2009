Visualizations

This folder contains scripts and documentation related to the analysis and visualization of the Neural Spike Sorter pipeline performance.

Scripts

The scripts subfolder holds specific Python files designed for tuning and verification.

peak_detection_analysis.py: Used to systematically test the detector model's performance (Precision, Recall, F1 Score) across a range of confidence thresholds on the labeled D1 dataset. It also provides a visual inspection of a signal segment to confirm spike detection quality.
